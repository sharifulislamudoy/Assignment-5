Question-1: What are the different ways to select on element in the DOM?

getElementById("id"), getElementsByClassName("class"), querySelector("selector"), querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML – Gets or sets the HTML inside an element. It parses HTML tags.innerText – Gets or sets the visible text inside an element. It ignores hidden text and doesn’t parse HTML.textContent – Gets or sets all text inside an element, including hidden text. Doesn’t parse HTML, but shows everything.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element instead of adding it to multiple child elements. The event bubbles up from child elements to the parent, where it is handled.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and then bubbles up through its parent elements until it reaches the root (document).

Question-5: How do you create, add, and remove elements using JavaScript?

createElement() → Creates, appendChild() / append() → Adds, remove() / removeChild() → Deletes
